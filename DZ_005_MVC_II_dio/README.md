    U .env fileu podesiti parametre za spajanje s bazom podataka:

        DB_CONNECTION=mysql
        DB_HOST=127.0.0.1
        DB_PORT=3306
        DB_DATABASE=ime_baze_podataka
        DB_USERNAME=root
        DB_PASSWORD=
    
    Pokrenuti komandu: php artisan key:generate 

    Pokrenuti komandu: php artisan serve
